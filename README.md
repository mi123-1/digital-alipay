# 数字支付宝网站版

这是数字支付宝的网站版，可部署到外部网络服务器上使用。

## 部署步骤

1. 将 `website` 目录上传到您的网络服务器根目录
2. 确保服务器支持静态文件访问
3. 访问服务器地址即可使用

## 主要功能

- 钱包余额管理
- 余币宝转入转出
- 交易账户管理
- 价格走势图
- 转币功能

## 技术说明

- 使用 Vue 3 框架
- 使用 localStorage 进行本地存储
- 响应式设计，支持移动端和桌面端

## 注意事项

- 网站版使用 localStorage 存储数据，数据仅保存在当前浏览器中
- 为了安全起见，建议在 HTTPS 环境下使用
- 网站版功能与 App 版基本一致，但可能存在一些细微差异

## 目录结构

```
website/
├── components/          # 组件目录
├── pages/               # 页面目录
├── static/              # 静态资源目录
├── utils/               # 工具函数目录
├── App.vue              # 应用主组件
├── index.html           # 入口 HTML 文件
├── main.js              # 应用入口文件
├── pages.json           # 页面配置文件
└── README.md            # 说明文档
```
